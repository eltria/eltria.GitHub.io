hugo配置

  1. 安装hugo
1

brew install hugo

这边用的是mac的homebrew,如果是其他系统的话找各自的方法

  1. 新建hugo文件夹
1

hugo new blog

新建好的文件夹跟这个差不多

  1. 安转主题
1

git clone https://github.com/olOwOlo/hugo-theme-even themes/even

这边用的例子是even主题,推荐的朋友挺多的

1
2

cp -rf themes/even/exampleSite/content .
cp themes/even/exampleSite/config.toml .

这边可以直接偷懒,把例子拿来直接用

#!/bin/bash
# 部署到 github pages 脚本
# 错误时终止脚本
set -e

# 删除打包文件夹
rm -rf public

# 打包。even 是主题
hugo -t even # if using a theme, replace with `hugo -t <YOURTHEME>`

# 进入打包文件夹
cd public

# Add changes to git.

git init
git add -A

# Commit changes.
msg="building site `date`"
if [ $# -eq 1 ]
  then msg="$1"
fi
git commit -m "$msg"

# 推送到githu  
# nusr.github.io 只能使用 master分支
git push -f https://github.com/eltria/eltria.GitHub.io.git master

# 回到原文件夹
cd ..

根据网上的一个朋友的脚本改的,核心的点是hugo -t even打包之后,每次都是git 强行覆盖master
